Projects I Did in School

1st Year: Tara Jeep

Description:

Tara Jeep is a fare calculation and management system designed for e-jeepney operations. The system helps commuters estimate fares based on distance, assists drivers in tracking daily revenue, and provides route management tools for administrators. By integrating real-time data, Tara Jeep enhances efficiency in urban transportation and promotes transparency in fare collection.

2nd Year (DSA): Library Management System

Description:

This Library Management System streamlines book lending, cataloging, and inventory management for academic and public libraries. It allows librarians to manage borrower records, track book availability, and generate reports on library usage. The system includes a user-friendly search function to help students and faculty locate books efficiently. Features such as overdue book alerts and automated fine calculations ensure smooth library operations.
